# Netflix Content Analysis Dashboard – Tableau Project

This project presents an interactive **Tableau dashboard** that explores Netflix’s global content library using data visualization techniques. The dashboard provides insights into content distribution, genre trends, ratings, release patterns, and country-wise availability.

1. # Project Overview
The dashboard analyzes Netflix Movies & TV Shows dataset to uncover:
* 🎬 Distribution of Movies vs TV Shows
* 🌍 Country-wise content availability (World Map Visualization)
* 📅 Year-wise growth of Netflix content
* 🎭 Top 10 Genres by total titles
* 🔞 Rating-wise classification of content
* ⏱ Duration, Release Year & Date Added insights
* 📖 Dynamic title description view

2. # Key Insights
* Movies dominate the platform compared to TV Shows.
* Significant content growth observed after 2015.
* Documentaries and Stand-Up Comedy are among the most frequent genres.
* Content distribution varies significantly across countries.

3. # Tools & Technologies Used
* **Tableau Desktop** – Data Visualization & Dashboard Design
* Data Cleaning & Preparation
* Interactive Filters & Parameters
* Map Visualizations
* Bar Charts, Pie Charts, Area Charts

4. # Dashboard Features
* Interactive filters (Type, Title)
* Country-level content analysis
* Rating distribution breakdown
* Year-wise content trend visualization
* Genre-based comparison

5. # Dataset
The dataset contains information such as:
* Title
* Type (Movie / TV Show)
* Country
* Genre
* Rating
* Duration
* Release Year
* Date Added

6. # Objective
To transform raw Netflix dataset into meaningful business insights using visual analytics and storytelling techniques.
